# 연방 준비 제도  (**Fed**eral Reserve System)



![[fed_structure.png]]

## 1. 정의

```
연방 가지의 12곳에 산재한 연방준비은행으로 구성되어, 중앙은행이 아닌 민간 은행 연합체 성격을 가짐
사실상 독립적인 사립은행들의 모임
```

## 2. 임무
- 통화정책 수행(Monetary Policy)    
    - 경제가 너무 뜨겁거나(인플레이션) 너무 식거나(경기침체) 하지 않도록 조정해. 구체적으로 **“최대고용 + 물가안정 + 중기적 완만한 금리”라는 목표**.
    - 이를 위해 기준금리 설정, 공개시장조작(Open Market Operations), 준비금 제도, 할인창구 등 다양한 도구를 사용. 

- 정부의 은행 기능(Banker to the Government)    
    - 미국 재무부(Treasury)가 연준과 협력해 정부의 계좌관리, 국채발행·상환, 재무서비스 등을 수행해
     
## 3. 권한

- 기준 금리 설정  (최대 고용 + 물가 안정)

- 정부의 은행 역활 (재무부 계좌) 

- 유동성 공급/흡수 수단 보유 
	- 공개 시장운영 (Open Market Operations) : 국채, 회사채 매도/ 매수 -> 현금량 조절
	- 역 레포 (Overnight Reverse RePo) : 하루동안 시중 은행에 채권을 매도 -> 유동성 풍부시 단기 금리 조절 역활 ( 역 레포 금리)
	
- 시중 은행들의 **지급 준비금** 계좌 보유 


```ad-info
title:지급준비금 (Reserve for payment)
고객들이 언제든 돈을 찾을 수 있도록, 예금중 일부를 중앙은행에 의무적으로 예치해야하는 금액

지급 준비율에 따라 필요 지급 준비금과 초과 지급 준비금으로 구분시켜, 이자율을 다르게 지급했지만, 코로나 이후 지급 준비율을 0%로 인하하고 현재까지도 유지되고 있음

그래서 필요/초과를 구분할 필요가 없어지며, 이자도 IOER(Interest On Reserve Balance)로 통일됨

```

## 4. 수입/지출

### (1) 수입
- 가지고 있는 채권 이자 수입 ( 국채가 가장큼)

### (2) 지출
- 대출 및 채권 매매

# 2. 연방준비제도 이사회 
(Federal Reserve Board of Governors)
- 연방공개시장위원회(FOMC)는 공개시장조작 정책을 수립·집행하는 기구
- 연방준비제도이사회(FRB) 8명 + 뉴욕 연방 은행 총재 (1명) + 그외 은행 총재 (4명)  = 총 12명

# 3. 공개 시장 조작 (Open Market Operations)

```ad-info
title: 공개 시장 조작(Open Market Operations)
- 통화량 조절이 필요시, 공개적으로 국공채를 매매해, 인위적으로 시장의 통화량을 조절
- 시장에 유통되는 자본의 속도와 방향성에 변화를 만들고  모든 자산군의 가격에 영향을 미침
```

- FOMC에서 통화 정책에 대해 의사결정을 내리고, 실제 공개시장 운영은 뉴욕 연준이 실행

## (1) 양적 완화 (Quantitative Easing)
국채, MBS등을 시장에서 사들여서 시장에 유동성을 푸는 행위

- 기준 금리가 0%인 상태에서 더 이상 금리를 내릴 수 없을때 사용됨
- 기존의 단기물을 이용해 유동성을 조절하던 방식과 달리, **장기물 국채를 직접 매입**하는 방법으로 유동성 공급

## (2) 양적 긴축 (Quantitative Tighten)
보유하는 자산에 대해 재투자 하지 않거나, 유통시작에 매각해, 시중의 유동성을 흡수

- 소극적 : 만기가 도래하는 국채 및 MBS에 재투자 하지 않음
- 적극적 : 만기가 안된 국채 및 MBS를 강제로 매작

## (3) 질적 완화 (Quantitative Easing)
- 양적완화 이후에도 유동성 공급이 필요할 경우, **주식 이나 회사채 채권**까지 매입해서 유동성을 공급한다.


```ad-info
title: 유동성 공급 순서
기준금리 인하 -> 양적완화 -> 질적 완화
```

# 4. 연준의 유동성 조절 계좌

## (1) System Open Market Account (SOMA)
- 연준이 보유하고 있는 증권을 관리하는 계좌
- 공개 시작 조작으로 매매한 미국 국채, 정부기관채, 주택저장증권(MBS)을 보유

```ad-question
title: 예시
1. 양적 완화 시, 시장의 국채를 연준이 매입하면, 연준의 SOMA 계정의 보유자산이 증가
2. 양적 긴축 시, 보유중인 국채가 만기때, 재투자하지 않아 국채비중이 줄어 보유자산이 하락
```

즉 SOMA 잔고와 시중 유동성은 양의 상관관계가 있다. 
- 급격히 떨어지면, 양적 완화 실행
![[SOMA_SNP.png]]



## (2) 대출
: 연준은 각 금융기관에게 필요시 사용할 수있는 대출 창구를 운영

### 재 할인 창구 (Discount Window)
: 단기 유동성 부족을 겪는 은행에 담보 제공받고 대출

### PPPLF(Paycheck Protection Program Liquidity Facility):
: 코로나 19 대응으로 2020년 도입되어, 은행들이 중소기업에 제공한 PPP대출을 담보로 연준으로부터 대출

### BTFP (Bank Term Funding Program)
: 2023년 SVB 사태 이후, 은행들이 보유한 국채, MBS 등을 액면가로 담보 취급해 대출

![[Loan.png]]


## (3) 연준의 지급 준비금
: 지금 준비금이란 은행이 예금주의 인출에 대비해 중앙은행에 예치해야 하는 자금.

```ad-question
title: 예시
이론적으로는 지급준비금의 증가/감소가 은행 대출 증가/감소로 이어져 유동성을 증가/감소 시킨다. 즉, 지급준비금과 시중 유동성은 **양의 상관관계**를 가집니다.
```


![[reserve.png]]

## (4) 재무부의 TGA
: 연방 정부의 재무주가 운영하는 계좌로, 정부 수입과 지출이 관리되는 계좌

```ad-question
title: 예시
1. 재무부가 발행한 국채를 은행이 매입하면, TGA 잔고는 늘어나고, 은행 지급 준비금은 감소.(음의 상관관계)
2. 재무부가 재정 지출을 통해 TGA 자금을 방출하면 지급 준비금이 증가
   
즉, TGA잔고가 늘어나면 유동성이 줄어들고, TGA 잔고가 줄어들면 유동성이 증가
```

![[TGA.png]]


## (5) 역레포 잔고

```ad-info
title: Repo
일종의 단기 대출로, 가지고 있는 증권을 담보로 팔아서 대출을 받고, 나중에 증권을 다시 살 것을 약속하는 거래

주로 은행이나 금융기간이 단기 자금을 조달 할때 사용

Repo는 거의 금융위기가 아니면, 잘 사용되지 않는다. 연준에게 까지 와서 돈을 땡길 정도면 어지간히 위기인것이다.

```

```ad-info
title: Overnight Reverse Repo (RRP)
레포 거래의 반대로, 시중은행들의 하루정도의 단기로 증권을 담보로 받고, 대출을 해주고, 나중에 다시 팔 것을 약속하는 거래

주로, 중앙은행이 유동성을 흡수 할때 사용.

```

- 연준이 시중의 과도한 유동성을 흡수하고, 단기 금리의 하한선을 유지하는데 도움을 준다.
- 역레포 잔고가 증가한다는 것은 연준이 더 많은 유동성을 흡수하는 중이라고 해석

![[RRP.png]]

## (6) 지표별 상관관계

```ad-info
title: 지표 별 연관성
SOMA  + 대출 = 지급 준비금 + 역레포 계좌 + 재무부 TGA
지급 준비금 = SOMA + 대출 - 역레포 - TGA

지급 준비금이 유동성과 가장 강력한 양의 상관관계
```


```ad-attention
title: 순 유동성 지표
순 유동성 : 연준 대차 대조표 - TGA + 역레포
```

![[total.png]]
	- SOMA(보라)는 양적 긴축이 시작된 22년 6월 기점으로 국채 보유량을 축소하니깐 계속해서 감소
	- 연준의 대출(노랑)은 SVB 사태 2023년 3월에 급증이후 다시 조금씩 줄어듬
	- TGA 잔고(빨강)은 바이든 시기에 재정 지출을 계속해 꾸준히 감소세 유지후, 2023년 여름에 부채 한도 위기를 한번 겪고, 세금들어오면서 한번 더 증가. 그리고 선거때문에 재무부가 유동성 공급
	- 역레포 (초록)은 시중에 유동성이 넘쳐나서 역레포 잔고로 몰렸다가, 2023년 하반기 양적긴축으로 꾸준히 하향세를 보임

# 5. 매커니즘


![[Machanism|800]]

## (1). 재무부 지출 & 국채 발생
- 재무부가 인프라, 보조금등 정부 지출을 하면 :  TGA 잔고-> 개인/기업 계좌 -> 지금 준비금 계좌
- 국채 발행 시: 국채 발생 -> 은행 매입 -> 지급 준비금 계좌 출금 -> TGA 잔고로 이동

## (2). 연준의 양적 완화 시
-  연준이 시장의 채권을 매입 -> 은행에게 유동성 지불 -> 지급 준비금 계좌 증가 -> SOMA 계좌 증가

## (3) 양적 긴축시
- 연준이 채권을 재투자 하지 않으면 -> 재무부에서 금액 지불 -> TGA 잔고 감소 or 새로운 국채 발행후 상환 -> 은행이 국채 매각 -> 지급 준비금 감소
