

- 태국은 우리나라와 반대로 요일 - 일자 - 월 - 연도 순으로 표기한다.

# (1) 요일 표현
- 완 : 일
- 별의 의미가 있다.
-> *하지만 완은 요일을 말하고, 완 티 가 몇번째 날 해서 일자를 말한다.*

| *한국어* | *태국어*        | *기타* | *발음*          |
| ------ | ------------ | ---- | ------------- |
| 월요일    | 완 짠(/)       | 짠: 달 | ![[1.요일.m4a]] |
| 화요일    | 완 앙캉(-/)     |      |               |
| 수요일    | 완 풋(-)       |      |               |
| 목요일    | 완 파르핫(--/)   |      |               |
| 금요일    | 완 숙(-)       |      |               |
| 토요일    | 완 싸우(v)      |      |               |
| 일요일    | 완(/) 아팃(\\/) |      |               |
|        |              |      |               |

# (2) 일 표현

```ad-info
일자는 서수로 표현한다 -> 티(^)만 붙이면된다.
티: 몇번째날같은 뜻

티(^) 능 (1일)
티(^) 썽 (2일)

```
