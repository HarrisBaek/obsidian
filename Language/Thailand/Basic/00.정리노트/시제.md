

## 🌈 태국어 시제·상태 표현 완성표 ver.2

| 구분                            | 의미 / 시점                   | 예문 (남성 공손형)                                      | 한국어 번역              |
| ----------------------------- | ------------------------- | ------------------------------------------------ | ------------------- |
| 🕒 **현재 (일반)**                | 평상시 / 반복되는 일              | ผมกินข้าวทุกวันครับ  <br>(폼 낀 카오 톡 완 크랍)         | 저는 매일 밥을 먹어요.       |
| 🕐 **현재진행① (กำลัง)**          | 지금 ~하고 있는 중 (행동 중심)       | ผมกำลังกินข้าวครับ  <br>(폼 감랑 낀 카오 크랍)           | 지금 밥 먹는 중이에요.       |
| 🕑 **현재진행② (อยู่)**           | ~하고 있는 상태 (상태 지속)         | ผมกินข้าวอยู่ครับ  <br>(폼 낀 카오 유 크랍)             | 밥 먹고 있어요.           |
| 🕓 **현재진행③ (กำลัง ... อยู่)** | 한창 ~중이다 (행동+상태 강조)        | ผมกำลังกินข้าวอยู่ครับ  <br>(폼 감랑 낀 카오 유 크랍)     | 지금 한창 밥 먹는 중이에요.    |
| ⏳ **진행 직전 (하려던 참)**           | 막 ~하려는 참 / 곧 ~할 예정        | ผมกำลังจะไปซื้อน้ำครับ  <br>(폼 감랑 짜 빠이 쓰ือ 남 크랍) | 마침 음료를 사러 가려던 참이에요. |
| ⏮️ **과거 (행동 발생)**             | 과거에 실제 행동이 일어남            | ผมได้ไปตลาดครับ  <br>(폼 다이 빠이 딸랏 크랍)             | 저는 시장에 갔어요.         |
| ✅ **과거 (완료)**                 | 이미 완료된 일 / 결과 강조          | ผมกินข้าวแล้วครับ  <br>(폼 낀 카오 래오 크랍)            | 밥 먹었어요.             |
| ✅ **과거 (행동+완료 강조)**           | 행동 + 결과 모두 강조 (“드디어 ~했다”) | ผมได้กินข้าวแล้วครับ  <br>(폼 다이 낀 카오 래오 크랍)      | 드디어 밥을 먹었어요.        |
| 🚫 **부정 과거**                  | ~하지 못했다 / ~안 했다           | ผมไม่ได้ไปครับ  <br>(폼 마이 다이 빠이 크랍)              | 저는 못 갔어요.           |
| 🔜 **미래 (예정)**                | ~할 것이다 / 계획               | ผมจะไปตลาดครับ  <br>(폼 짜 빠이 딸랏 크랍)               | 저는 시장에 갈 거예요.       |
| ❌ **미래 부정**                   | ~하지 않을 것이다                | ผมจะไม่กินข้าวเย็นครับ  <br>(폼 짜 마이 낀 카오 옌 크랍)   | 저녁 안 먹을 거예요.        |
| 🕗 **아직 안 함 (미완)**            | 아직 ~하지 않음 / 진행 전          | ผมยังไม่กินข้าวครับ  <br>(폼 양 마이 낀 카오 크랍)        | 아직 밥 안 먹었어요.        |
| 🕘 **이미 함 (완료)**              | 이미 끝남 (완료 상태 유지)          | ผมได้ทำงานแล้วครับ  <br>(폼 다이 탐응안 래오 크랍)         | 이미 일했어요.            |
