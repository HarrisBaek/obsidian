## 1. 변수 (read / write)



```ad-note
title: variable types
Immutable
data type : primitive types, frozen objects (i.e. object.freeze())

Mutable data types: allobjects bydefault are mutable in JS
favor immutable data type always for a few reasons
```


변수는 총 3가지 단계를 통하여 사용되어진다.
1. 선언단계
2. 초기화 단계
3. 할당 단계

#### 01. var (don't ever use this!!)
Hoisting 때문에 var를 사용하면 비정상적인 사용을 가능하게 하기때문에 ==더이상은 사용하지 않는다==.
- 블러 스코프가 적용이 안됨
- 1. 선언 및 초기화 단계(undefined) 가 동시에 이뤄진다.
- 2. 그 다음 할당된다.

```javascript
function add(num1, num2){
	var result = num1+num2;
}

add(2,3);
console.log(result); //ReferenceError
//var도 선언이 함수내에서 선언이면, 바깥이면 사용불가 (유일하게 함수 스코프는 벗어날수없다)


```

#### 02.let (Mutable Type)
- 지역 변수의 개념으로 블록 스코프를 가진다.
- 1. 선언 단계 , 2. 초기화 단계, 3. 할당 단계가 따로 되어 호이스팅되어도 error발생


```javascript
'use strict';
console.log('Hello World');

{
	let name='ellie';
	console.log(name);
	name = 'hello';
	console.log(name);
}


console.log(name);
```

#### 03.Constant (Immutable Type)
상수로 선언된 이유로는 값을 변경할 수 없다.
- 선언 초기화 할당이 동시에 이뤄진다.
- Security
- Thread safety
- reduce human mistakes

```javascript
const daysInWeek = 7;
const maxNumber = 5;
```
