- **기간**: 약 1년 (52주)
    
- **모듈 구성**: 총 5개의 메인 단계
    
    1. 기초 모듈 (Weeks 1-12)
        
    2. 기술 심화 모듈 (Weeks 13-32)
        
        - 여기에는 각각의 기술 + Supabase 포함

## 추가 팁

- Supabase를 배우면서 **데이터 모델링**과 **권한 설계(RLS: Row Level Security)**도 꼭 다뤄야 해 — Supabase는 PostgreSQL 기반이며, RLS를 통해 사용자별 데이터 필터링 가능. [tools.dev+1](https://blog.boldtech.dev/what-is-supabase-vs-firebase/?utm_source=chatgpt.com)
    
- 각 모듈 간 기술 연결고리를 의식하자. 예컨대: Supabase에서 데이터를 받아오면 TanStack Query로 캐싱 → Zustand로 클라이언트 상태관리 → UI는 Tailwind/shadcn/ui로 구성 → 필요시 Framer Motion 애니메이션 적용
