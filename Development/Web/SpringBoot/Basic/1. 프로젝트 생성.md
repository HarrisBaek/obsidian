### 기본 컨트롤러 생성

1. Controller 및 Html 작성

- 컨트롤러에서 리턴 값으로 문자를 반환하면 뷰 리졸버( viewResolver )가 화면을 찾아서 처리한다. 스프링 부트 템플릿엔진 기본 viewName 매핑

`resources:templates/ +{ViewName}+ .html`

```java
package com.example.hello.hellospring.contoller

//모든 컨트롤러는 이렇게 적어 놔야 함
@Controller
  public class HelloController {

	//HTTP - Get 
    @GetMapping("hello")
    public String hello(Model model) {
		// (attribute Name, attribut Value)
        model.addAttribute("data", "hello!!");

		//return hello.html  (Model에 attrribute를 심어서 보냄)
        return "hello";
    }
}
```

```html
<!DOCTYPE HTML>

<html xmlns:th="http://www.thymeleaf.org">
  <head>

      <title>Hello</title>

      <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
  </head>

<body>  
	/* controller에서 attribute로 받은거 대입 **/
	<p th:text="'안녕하세요. ' + ${data}" >안녕하세요. 손님</p>
</body>
</html>

```
