### Postgresql 접속

- 기본적으로 postgres라는 super user가 생성되어져 있다.
- 아래 명령어를 통해서 postgres로 로그인한다
```base

//postgres로 로그인
psql postgres 

//현재 ROLE보기
postgres= \du

//ROLE 생성 - CREATEDB 권한을 추가
postgres=> ALTER ROLE testdb CREATEDB;

//유저 생성
createuser testdb --createdb

//생성된 유저로 로그인
psql postgres -U testdb;

postgres=> CREATE DATABASE testdb;

//Db list 조회
postgres=> \list  

```
